CRUD methods and supported URLs:

CRUD method
 Explanation
GET /api/authors
 Retrieves a list of all authors with their data (name and surname)
GET /api/authors/<author_id>
 Retrieves information (name, surname) about author with the author_id
GET /api/quotes
 Retrieves a list of all quotes with its data (message and author id who stated this message or null if the author is unknown or deleted)
GET /api/quotes/<quote_id>
 Retrieves information about quote with the id quote_id
GET /api/authors/<author_id>/quotes
 Retrieves all quotes stated by the person with the author_id
GET /api/authors/<author_id>/quotes/<quote_id>
 Retrieves a quotes with id quote_id stated by the person with the author_id
POST /api/authors
 Creates new author entry (name and surname)
PUT /api/authors/<author_id>
 Updates information (name, surname) about author with the author_id
POST /api/quotes
 Creates an quote with its data (message and author id who stated this message or null if the author is unknown or deleted)
PUT /api/quotes/<quote_id>
 Updates information about quote with the id quote_id
POST /api/authors/<author_id>/quotes
 Creates or updates an quote stated by the person with the author_id
PUT /api/authors/<author_id>/quotes/<quote_id>
 Updates a quote with id quote_id stated by the person with the author_id
DELETE /api/authors/<author_id>
 Deletes an author entry with the author_id
DELETE /api/quotes/<quote_id>
 Deletes a quote entry with the id quote_id
DELETE /api/authors/<author_id>/quotes/<quote_id>
 Deletes a quote with id quote_id stated by the person with the author_id
